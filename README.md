# IOC_WBL_v10.2

The repository contains five tables with lists of birds taxa of the world fauna. These tables are designed for using bird taxonomy in a databases. Each of the tables contains a list of one of the five levels of bird taxa (order, family, genus, species, subspecies). The IOC World Bird List V. 10.2 was used for create the tables (Gill F, D Donsker & P Rasmussen  (Eds). 2020. IOC World Bird List (v10.2). doi :  10.14344/IOC.ML.10.2).
